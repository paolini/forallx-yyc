\problempart
\label{pr.TT.concepts}
Answer each of the questions below and justify your answer.
\begin{compactlist}
\item Suppose that \metav{A} and \metav{B} are logically equivalent. What can you say about $\metav{A}\eiff\metav{B}$?
\item[] \myanswer{\metav{A} and \metav{B} have the same truth value on every line of a complete truth table, so $\metav{A}\eiff\metav{B}$ is true on every line. It is a tautology.}
\item Suppose that $(\metav{A}\eand\metav{B})\eif\metav{C}$ is neither a tautology nor a contradiction. What can you say about whether $\metav{A}, \metav{B} \therefore\metav{C}$ is valid?
\item[] \myanswer{Since the sentence $(\metav{A}\eand\metav{B})\eif\metav{C}$ is not a tautology, there is some line on which it is false. Since it is a conditional, on that line, \metav{A} and \metav{B} are true and \metav{C} is false. So the argument is invalid.}
\item Suppose that $\metav{A}$, $\metav{B}$ and $\metav{C}$  are jointly unsatisfiable. What can you say about $(\metav{A}\eand\metav{B}\eand\metav{C})$?
\item[] \myanswer{Since the sentences are jointly unsatisfiable, there is no valuation on which they are all true. So their conjunction is false on every valuation. It is a contradiction}
\item Suppose that \metav{A} is a contradiction. What can you say about whether $\metav{A}, \metav{B} \entails \metav{C}$?
\item[] \myanswer{Since \metav{A} is false on every line of a complete truth table, there is no line on which \metav{A} and \metav{B} are true and \metav{C} is false. So the entailment holds.}
\item Suppose that \metav{C} is a tautology. What can you say about whether $\metav{A}, \metav{B}\entails \metav{C}$?
\item[] \myanswer{Since \metav{C} is true on every line of a complete truth table, there is no line on which \metav{A} and \metav{B} are true and \metav{C} is false. So the entailment holds.}
\item Suppose that \metav{A} and \metav{B} are logically equivalent. What can you say about $(\metav{A}\eor\metav{B})$?
\item[] \myanswer{Not much. Since $\metav{A}$ and $\metav{B}$ are true on exactly the same lines of the truth table, their disjunction is true on exactly the same lines. So, their disjunction is logically equivalent to them.}
\item Suppose that \metav{A} and \metav{B} are \emph{not} logically equivalent. What can you say about $(\metav{A}\eor\metav{B})$?
\item[] \myanswer{\metav{A} and \metav{B} have different truth values on at least one line of a complete truth table, and $(\metav{A}\eor\metav{B})$ will be true on that line. On other lines, it might be true or false. So $(\metav{A}\eor\metav{B})$ is either a tautology or it is contingent; it is \emph{not} a contradiction.}
\end{compactlist}

\problempart
Consider the following principle:
	\begin{quote}
		Suppose $\metav{A}$ and $\metav{B}$ are logically equivalent. Suppose an argument contains $\metav{A}$ (either as a premise, or as the conclusion). The validity of the argument would be unaffected, if we replaced $\metav{A}$ with $\metav{B}$.
	\end{quote}
Is this principle correct? Explain your answer.
\\\myanswer{The principle is correct. Since $\metav{A}$ and $\metav{B}$ are logically equivalent, they have the same truth table. So every valuation that makes $\metav{A}$ true also makes $\metav{B}$ true, and every valuation that makes $\metav{A}$ false also makes $\metav{B}$ false. So if no valuation makes all the premises true and the conclusion false, when $\metav{A}$ was among the premises or the conclusion, then no valuation makes all the premises true and the conclusion false, when we replace $\metav{A}$ with $\metav{B}$.}
